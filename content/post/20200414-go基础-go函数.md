##  go 函数结构
   > func函数关键字, 函数名, 参数列表, 返回列表, 函数体

## 访问性
   > 大写时,其他包可访问
   > 小写时, 相同包可访问

## 实例
```go
func func_namen(param-list)(result-list){
	function-body
}
```

## 特点:
1. 函数可以没有输入参数,也可以没有返回值(默认返回0)
2. 多个相邻的相同类型参数可以简写
3. 支持有名的返回值
4. 不支持重载
5. 不支持嵌套
6. 不支持默认参数
